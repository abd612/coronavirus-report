# Coronavirus Report

A website created using Spring Boot, Thymeleaf and Bootstrap, to view the latest statistics for COVID-19, updated on daily basis.

## Description

* Languages: Java 8, HTML, CSS, JavaScript
* Frameworks: Spring Boot
* Libraries: Thymeleaf, Bootstrap
* Databases: MongoDB, PostgreSQL
* Website: https://coronavirus-report.herokuapp.com/

## Features

* Website with complete front-end
* REST API
* MVC Architecture
* Microservices
* Spring Web
* Spring Thymeleaf  
* Spring Data with MongoDB
* Spring Data with PostgreSQL  
* Spring Security
* Springfox Swagger
* Bootstrap
